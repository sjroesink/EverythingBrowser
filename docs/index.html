Connections

Connect to anything

Seamlessly manage files across all your storage backends from one unified interface.

SFTP

Securely connect to remote servers via SSH. Supports password, key-based, and agent authentication.

Backblaze B2

Browse and manage your B2 cloud storage buckets using the S3-compatible API.

Docker Volumes

Browse and manage files inside Docker volumes directly without spinning up containers manually.

Docker Exec

Access the filesystem of running Docker containers. Navigate, upload, and download files live.

Local Filesystem

Quick access to folders on your machine. Useful as a source or target for transfers.

Features

Built for power users

A modern file manager with the flexibility and keyboard shortcuts you expect.

Dockable Multi-Pane Layout

Split your workspace into panes — horizontally or vertically. Drag tabs between panes with VS Code-style docking indicators.

Multi-Tab Interface

Open multiple connections side by side. Each pane supports its own set of tabs with full drag-and-drop support.

Detached Windows

Pop tabs out into separate windows. Tabs reattach automatically when the window is closed.

Keyboard Shortcuts

Navigate, split panes, switch tabs, and manage connections — all from the keyboard. Ctrl+B toggles the sidebar, Ctrl+W closes tabs.

Drag & Drop Transfers

Upload files by dropping them into the browser. Transfer files between connections with progress tracking.

Secure Credentials

Passwords and keys are stored in your OS keyring — never in plain text config files.

Under the Hood

Built with Rust & React

Native performance with a modern, responsive UI.

Tauri v2

Lightweight native shell. No Electron bloat.

Rust Backend

Async I/O with Tokio. Memory-safe and fast.

React 19

Modern hooks, Zustand state, Tailwind styling.
